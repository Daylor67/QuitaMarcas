## Acerca de este proyecto

Versión derivada e independiente de [SmartStitch](https://github.com/MechTechnology/SmartStitch) por [MechTechnology](https://github.com/MechTechnology).

Esta versión agrega funcionalidad de **eliminación de marcas de agua** al proyecto original.

**No es un fork oficial** - se mantiene de forma independiente.
